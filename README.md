# 💳 Banking Management System (PHP + MySQL)

## 📌 Overview
A web-based banking system that allows users to manage accounts, perform transactions, and view history. Admin can monitor users and operations.

## 🚀 Features
- User login & registration
- Money transfer & withdrawal
- Transaction history
- Admin dashboard

## 🛠️ Tech Stack
PHP | MySQL | HTML | CSS | WAMP

## ⚙️ How to Run
1. Copy project to WAMP www folder
2. Import database.sql in phpMyAdmin
3. Run on localhost
